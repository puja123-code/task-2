### Laundry Wallah / Laundry Mart 🧺
Laundry Wallah / Laundry Mart is a simple HTML-based webpage designed to showcase laundry services.
The page provides information about available services, pricing, and allows customers to book services online

## Feature
Clean and simple user interface

List of laundry services

Price list for services

Booking form for customers

Contact information section

### Technologies Used
HTML5

Basic CSS (if added)

Form elements for booking

### Sections in the Webpage
1. Welcome Section

Introduces the Laundry Wallah / Laundry Mart service and explains the purpose of the website.
